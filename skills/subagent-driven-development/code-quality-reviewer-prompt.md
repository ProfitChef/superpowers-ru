# Шаблон промпта для ревьюера качества кода

Используйте этот шаблон при запуске субагента-ревьюера качества кода.

**Назначение:** Убедиться, что реализация сделана качественно (чистый, протестированный, поддерживаемый код)

**Запускайте только после того, как проверка соответствия спецификации пройдена.**

```
Инструмент Task (superpowers:code-reviewer):
  Используйте шаблон из requesting-code-review/code-reviewer.md

  WHAT_WAS_IMPLEMENTED: [из отчёта исполнителя]
  PLAN_OR_REQUIREMENTS: Задача N из [файл-плана]
  BASE_SHA: [коммит до задачи]
  HEAD_SHA: [текущий коммит]
  DESCRIPTION: [краткое описание задачи]
```

**Ревьюер кода возвращает:** Сильные стороны, проблемы (критические/важные/незначительные), заключение
